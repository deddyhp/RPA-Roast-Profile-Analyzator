# RPA — Roast Profile Analyzator

Local Streamlit application for roast profile records and coffee knowledge.

## V0.3.0

- Existing migrated roast database
- Artisan `.alog` import and profile viewer
- Coffee Knowledge Library with image, category, tags, source, and notes
- Seven initial coffee-processing infographics

## Run

```bash
py -m streamlit run app.py
```

Operational database, Artisan files, and knowledge images remain local and are excluded from GitHub.
